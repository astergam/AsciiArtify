## Демонстрація роботи програми в середовищі Kubernetes  
Створюємо псевдоним для команди kubectl  
```bash
alias k=kubectl
```  
Перевіряємо наявні сервіси  
```bash
k get svc
```  
Активуємо порт для сервісу ambassador  
```bash
k port-forward -n demo svc/ambassador 8088:80
```
